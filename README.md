## 스터디 진행 방식

**스터디원**
- [김가경](https://github.com/GaGa-Kim)
- [최가희](https://github.com/cutehumanS2)

**스터디 방식**
- 교재 목차에 따라 스터디를 진행 후 커밋 및 학습 보고
- 이후 백준 문제 풀이
<br/>

**스터디 목차**
> 1. 그리디
> 2. 구현
> 3. DFS % BFS
> 4. 정렬
> 5. 이진 탐색
> 6. 다이나믹 프로그래밍
> 7. 최단 경로
> 8. 그래프 이론
> 9. 기타 알고리즘
